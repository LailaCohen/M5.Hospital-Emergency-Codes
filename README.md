# Hospital Emergency Codes
In this lesson I practiced using enum. I really enjoyed doing it. I did not have a lot of problems, only a few small ones.
## What I Did
* Created a fixed list of emergency codes for hospitals.
* Added all the codes with their meanings.
* Saved the meaning of each code.
* Used a constructor to assign values.
* Created a method to get the meaning of each code.
* Used a switch statement to display instructions for each code.
* Used Hospital.values() to automatically go through all the codes.
* Used desp(code) to run the instructions function for each code.
## Emergency Codes
* Code Red – Fire
* Code Blue – Cardiac Arrest
* Code Yellow – Bomb Threat
* Code Orange – Hazardous Material Spill
* Code Green – Evacuation
* Code Clear – Situation has been resolved
